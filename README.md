# mi-primer-repositorio
Creacion de mi primer repositorio
Este repositorio fue creado como parte de la práctica de la Unidad 4, cuyo objetivo es aprender a crear un repositorio público y organizar archivos utilizando GitHub.

Estructura del repositorio
mi-primer-repositorio/
│
├── datos_ventas/
│   └── ventas.csv
│
└── datos_clientes/
    └── clientes.csv

 Descripción de archivos
1. datos_ventas/ventas.csv
Contiene información simulada de ventas:
ID de venta,
Producto,
Cantidad,
Precio unitario,
Total por venta
2. datos_clientes/clientes.csv
Incluye datos simulados de clientes:
ID de cliente,
Nombre,
Correo,
Teléfono,
Ciudad

Estructura organizada según la guía
